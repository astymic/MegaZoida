# AI Agents Rules (Anti-Crazy Guidelines)

## Основные правила для ИИ агентов
1. **Фокус на задаче**: Агенты обязаны строго следовать описанию проекта (MegaZoida). Никаких отступлений, самовольных смен жанра, добавления неуместных фич (например, NFT, мультиплеера без запроса).
2. **Код работает — не трогай**: Если текущая механика работает корректно и масштабируемо, запрещается переписывать ее ради "чистоты эксперимента", если только это не ломает производительность.
3. **Стабильность**: Никаких бесконечных циклов в коде. Любой update() цикл должен быть оптимизирован.
4. **Контекст**: Каждый ИИ агент, начинающий работу с репозиторием, должен прочитать этот файл и план реализации.
5. **Сохраняй рассудок**: Не создавай излишне сложных абстракций. Если можно написать простую формулу для урона, пиши простую формулу (Base Attack * Level). Не раздувай архитектуру, это инди-проект.
6. **Никаких галлюцинаций**: Используй только те ресурсы, движки и библиотеки, которые уже заявлены в `implementation_plan.md`. Игнорируй "новые крутые фреймворки", вышедшие вчера.
