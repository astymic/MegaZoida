# Инструкция по сборке APK и тестированию MegaZoida (Android)

Поскольку MegaZoida теперь является полноценным 3D-проектом для мобильных телефонов, лучше всего использовать **Capacitor** для обертки нашего веб-приложения (JS/TS + HTML Canvas) в нативное Android приложение (APK).

Ниже приведена пошаговая инструкция.

## Шаг 1: Подготовка к Capacitor
Убедитесь, что у вас установлена [Android Studio](https://developer.android.com/studio) и скачаны необходимые SDK (рекомендуется Android SDK 33+).

В терминале корневой папки вашего проекта (там же, где находится `package.json`), выполните установку ядра Capacitor и CLI инструмента:
```bash
npm install @capacitor/core
npm install -D @capacitor/cli
```

## Шаг 2: Инициализация
Запустите инициализацию Capacitor в вашем проекте:
```bash
npx cap init
```
- **App name**: Введите `MegaZoida`
- **App Package ID**: Введите `com.chapa.megazoida` (или любой другой ваш домен)
- **Web asset directory**: Введите `dist` (именно сюда Vite собирает билд игры)

## Шаг 3: Добавление платформы Android
Вам нужно установить пакет для работы с Android:
```bash
npm install @capacitor/android
npx cap add android
```
Эта команда создаст папку `android/` в корне вашего проекта.

## Шаг 4: Сборка игры
Каждый раз перед тем как собрать APK, вам нужно сначала скомпилировать сам веб-проект:
```bash
npm run build
```
Эта команда заполнит папку `dist/` свежими файлами игры. 

Затем синхронизируйте их с Android-проектом:
```bash
npx cap sync
```

## Шаг 5: Тестирование на Эмуляторе или реальном телефоне
Чтобы запустить игру на Эмуляторе Android Studio или собрать финальный APK, просто откройте проект Android Studio из консоли:
```bash
npx cap open android
```

### В интерфейсе Android Studio:
1. Дождитесь пока Gradle завершит синхронизацию (полоска внизу экрана).
2. Сверху на панели выберите ваш реальный телефон (предварительно подключенный по USB с режимом разработчика) или скачайте образ эмулятора (Нажав `Device Manager` -> `Create Device`). Мы рекомендуем тестировать 3D игру **только на реальных устройствах**, так как эмуляторы часто не имеют нормального аппаратного ускорения видеокарты для `Three.js` и будут очень сильно тормозить.
3. Нажмите зеленую кнопку **Play (Run 'app')** сверху. Игра скомпилируется и запустится на телефоне!

### Шаг 6: Финальный релиз (Сборка .APK файла)
Если всё работает и вы хотите поделиться APK:
1. В Android Studio в верхнем меню выберите **Build** -> **Build Bundle(s) / APK(s)** -> **Build APK(s)**.
2. После короткого ожидания в правом нижнем углу появится попап с зеленой ссылкой `locate`.
3. Нажав на неё, откроется папка Windows, где будет лежать ваш готовый файл **`app-debug.apk`**. Это и есть ваша игра MegaZoida!
