# MegaZoida - Project Essence and Task

## Суть проекта (Project Essence)
MegaZoida — это мобильная игра в жанре "рогалик" (roguelike), идейный клон популярной игры Megabonk. Суть игры заключается в выживании героя на закрытой или бесконечной арене против постоянно наступающих волн врагов. Игрок управляет перемещением героя, в то время как атаки происходят автоматически. Собирая опыт с поверженных врагов, герой повышает свой уровень, получая возможности для перманентных улучшений (баффов).

## Главная задача (Core Task)
Создать полностью рабочую, увлекательную и сбалансированную игру с нуля. Игра должна поддерживать мобильные устройства (сенсорное управление, адаптивный дизайн) и включать следующие механики:
1. **Базовые характеристики**: HP, мана, уровень (на старте 1), скорость передвижения, скорость атаки, сила атаки, защита.
2. **Управление**: WASD + Space для ПК (и виртуальный джойстик/тапы для мобильных).
3. **Автоатака**: Герой сам атакует ближайших врагов.
4. **Прокачка (Level Up)**: За убийство мобов дается опыт. При каждом повышении уровня игрок выбирает 1 из 3 случайных баффов на всю игру.
5. **Эскалация сложности**: Со временем (или с каждым уровнем) враги становятся сильнее (больше HP, атаки) и их количество возрастает.
6. **Система оружия**: По карте разбросаны сундуки, открывающиеся за монеты. Доступно 12 видов базового оружия. У героя может быть максимум 6 слотов под оружие. При получении нового, слабое оружие заменяется по правилам приоритета.
7. **Боссы**: Каждые 5 уровней появляется усиленная версия мобов — Босс.

## Роль Инди-Разработчика
Проект ведется в роли Game Designer + Project Manager + Full Stack Developer. Задачи включают не только написание кода, но и создание "feel" игры — сочные анимации, правильный баланс, приятные эффекты (screen shake, damage numbers) и общая полировка игры до состояния хита.
